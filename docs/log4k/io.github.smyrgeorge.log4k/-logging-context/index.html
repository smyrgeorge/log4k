LoggingContext

data class LoggingContext(val tags: MutableTags = mutableMapOf(), val tracer: Tracer? = null, val parent: TracingEvent.Span? = null) : CoroutineContext.Element(source)

The LoggingContext class represents an element in the coroutine context that holds tracing and tagging information. It provides mechanisms for managing and tracking spans in a tracing context and supports execution within the scope of a specific tracing span.
