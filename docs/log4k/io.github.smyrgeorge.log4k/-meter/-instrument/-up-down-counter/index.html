class UpDownCounter<T : Number>(name: String, meter: Meter, unit: String?, description: String?) : Meter.Instrument.AbstractCounter<T>

Represents an UpDownCounter instrument used for recording numerical values that can increase or decrease. This class is a specialized type of AbstractCounter used within a Metering system.
