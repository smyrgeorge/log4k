OpenTelemetryAttributes

object OpenTelemetryAttributes(source)

The OpenTelemetry object provides a set of constants used for tracing and monitoring various parts of an application or system. These constants represent standard attributes that can be used to describe exceptions, HTTP requests and responses, database operations, and thread states.

https://opentelemetry.io/docs/specs/semconv/registry/attributes/
