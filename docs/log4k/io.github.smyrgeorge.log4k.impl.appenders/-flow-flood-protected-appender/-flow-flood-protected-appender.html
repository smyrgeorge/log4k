The period in milliseconds after which the burst state resets. Default value is 5000 milliseconds.

This class ensures that the flow of events does not exceed the specified rate and implements a burst logic to temporarily allow higher rates. If the rate exceeds the allowed limits, excess events are dropped, and a logging event is generated to warn about the dropped events.
